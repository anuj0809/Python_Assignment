I have used TF-IDF in order to summarize the content in the news articles.
Firstly I cleaned the dataset a bit by dropping the rows with null values and then split them in 90-10 ratio.
Now for process of summarising starts. It takes 9 steps to implement it:-
1. Firstly I tokenized the sentences instead of words.And give weight to these sentences. 
2. Then I created a frequency matrix of the words in each sentence.
3. Then I calculated TermFrequency and generated a matrix.
4. I then created a table for documents per words i.e. "How many sentences contain a word".
5. Next I calculated IDF for each word in a paragraph and generated a matrix.
6. I then calculated TF-IDF by multiplying TF and IDF matrix together and got the resultant matrix.
7. Scoring a sentence based on TF-IDF score of words in a sentence to give weight to a paragraph.
8. To find the threshold I calculated the average sentence score.
9. Summary generation - A sentence is selected if sentence score is more than average score. A multiplier is used with threshold value in order to determine the length of summary.
